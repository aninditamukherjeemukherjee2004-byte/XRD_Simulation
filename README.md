FCC X-Ray Diffraction Simulation
Overview

This project simulates the X-ray diffraction (XRD) pattern of a Face-Centered Cubic (FCC) crystal using Python.
The program calculates diffraction peak positions using Bragg's Law and determines peak intensities using structure factor calculations for an FCC lattice.

The simulation uses parameters close to Copper (Cu) with a lattice parameter of approximately 3.6 Å and Cu Kα radiation (λ = 1.5406 Å).
